## Quick Start

```bash
# Install the CLI
npm install -g @sniper.ai/cli

# Initialize SNIPER in your project
sniper init

# Run the full lifecycle
/sniper-discover   # Phase 1: Discovery
/sniper-plan       # Phase 2: Planning
/sniper-solve      # Phase 3: Story Creation
/sniper-sprint     # Phase 4: Implementation
```
